def entropy(val):
    if len(val) == 0:
        return 0
    return -np.sum([v * np.log2(v) for v in val if v > 0])

def entropy_of_dataset(df, target_col):
    target_vals = df[target_col].value_counts(normalize=True)
    return entropy(target_vals)

def information_gain(df, feature, target_col):
    base_entropy = entropy_of_dataset(df, target_col)
    feature_vals = df[feature].value_counts(normalize=True)
    weighted_entropy = 0
    for val in feature_vals.index:
        subset = df[df[feature] == val]
        subset_entropy = entropy_of_dataset(subset, target_col)
        weighted_entropy += feature_vals[val] * subset_entropy

    info_gain = base_entropy - weighted_entropy
    return info_gain

3 Information Gain¶

  • A - Entropy(x) = -( 1/8log_2(1/8) + 3/8log_2(3/8) + 4/8log_2(4/8)

  • B - Gini_Index(x) = 1 - ((1/8)^2 + (3/8)^2 + (4/8)^2)

  • C - When trying to determine the optimal split for the age feature. We first need to calculate the entropy on the entire set of target variables. We got 0.954 bits. From here we calculate information gain for each age that we want to evaluate, by subtracting the weighted entropy of each of our age groups groups from the entropy of our target set. For instance the information gain for ages like 21 and 29 are 0.745 and 0.955 respectively. However, the information gain for the age 26 was ( 1.406 - 0.143 = 1.263 ). The fact that age 26 had the highest information gain, makes it the most optimal place to split.

4 Decision Tree Error Pruning¶

When a decision is to be made about pruning a sub tree or not, we need to evaluate if replacing that particular sub tree with just a singular node will result in fewer classification errors. Replacing the sub tree with a single node will end up predicting whatever the majority of the parent nodes classifications were. If this operation leads to the same or better classifications, then it will be "pruned" (replaced by a single node). This essentially simplifies the tree by having less splits. However, if the classifications end up being worse due to pruning, then we will leave that subtree intact since it provides better results. The pruning process of our particular tree is shown below.
